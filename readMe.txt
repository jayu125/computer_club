**파일 설명**

공격 수단 : 플레이어와 적들의 기본 공격(평타), 스킬 등의 클래스를 모아놓음.
음악 : 게임에 사용될 음향 효과, BGM(브금)을 모아놓음
폰트 : 게임에 텍스트를 표시할 때 사용될 폰트를 모아놓음
image : 게임에 사용될 모든 이미지들를 모아놓음
game.py : 게임의 중추. 게임 시스템 돌아가게 하는 파일.
player.py : 플레이어 클래스를 관리하는 파일.



**자주 쓰이는 용어 설명**

width : 너비, 가로폭
height : 높이, 세로폭

aa : auto attack 의 약자로 한국어론 "평타", "기본 공격".
